# StronaFiz - Interaktywna Strona Edukacyjna Fizyki

**StronaFiz** to interaktywna platforma edukacyjna dedykowana nauczaniu fizyki poprzez praktyczne symulacje i wizualizacje.

## 📋 Struktura Projektu

```
StronaFiz/
├── index.html              # Strona główna
├── css/
│   └── styles.css         # Style CSS
├── js/
│   └── main.js            # Główny plik JavaScript
├── simulations/           # Foldery z poszczególnymi symulacjami
│   ├── projectile.html
│   ├── pendulum.html
│   ├── waves.html
│   └── magnetism.html
├── assets/                # Grafiki i zasoby
└── README.md              # Ten plik
```

## 🎯 Główne Funkcje

- **Interaktywne Symulacje** - Wizualne demonstracje zagadnień fizycznych
- **Responsywny Design** - Dostęp z różnych urządzeń
- **Edukacyjne Materiały** - Wyjaśnienia i temat fizyki
- **Łatwe w Użyciu** - Intuicyjny interfejs

## 🚀 Jak Zacząć

1. Otwórz `index.html` w przeglądarce
2. Przejrzyj dostępne symulacje
3. Eksploruj interaktywne demonstracje

## 📚 Dostępne Symulacje

### 1. Ruch Projectyla
Obserwuj, jak obiekty poruszają się pod wpływem grawitacji. Zmień kąt i prędkość początkową.

### 2. Wahadło Proste
Badaj periodyczne ruchy wahadła i jak długość wpływa na okres drgań.

### 3. Fale Dźwiękowe
Zrozum naturę fal, interference i propagację dźwięku.

### 4. Magnetyzm
Eksploruj siły magnetyczne i interakcje między polami magnetycznymi.

## 🛠️ Technologie

- **HTML5** - Struktura strony
- **CSS3** - Stylowanie i responsywność
- **JavaScript (Vanilla)** - Interaktywność i symulacje
- **Canvas API** - Rysowanie i animacje
